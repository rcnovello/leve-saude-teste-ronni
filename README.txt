
Projeto com dados estáticos.

 - npm install para instalar todas as dependencias.
 - npm run build para compilar o código.
 - npm run start para iniciar servidor local.
 - Usando o Postman pode-se testar as rotas e obter os retornos esperados 
   Rota GET: http://localhost:3000/dev/agendas
   Rota POST: http://localhost:3000/dev/agendamento
